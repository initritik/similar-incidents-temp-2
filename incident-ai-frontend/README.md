# Incident AI Frontend

Initial React, TypeScript, Vite, Tailwind CSS, and shadcn/ui foundation for the Incident AI Assistant.

Vite is used for fast local development and a simple production build path. shadcn/ui is configured through `components.json`, Tailwind CSS variables, and the `@/` path alias so future enterprise UI components can be added consistently.

Run locally after installing dependencies:

```bash
npm install
npm run dev
```
